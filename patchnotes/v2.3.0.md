## 🔧 Technische Architektur

```

```

## 📈 Export-System-Verbesserungen

### 📊 **Team-basierte Organisation**
**Vorher (v2.2):**
```
bundesliga_2023-24.xlsx
├── Alle_Spiele (Sheet)
    ├── Bayern München vs Dortmund
    ├── Hamburg vs Bremen
```

**Jetzt (v2.3):**
```
bundesliga_2023-24.xlsx
├── Bayern_München (Sheet)

│   ├── vs Hamburg (Auswärts)
│   └── ...
├── Borussia_Dortmund (Sheet)
│   ├── vs Bayern (Auswärts)
│   ├── vs Bremen (Heim)
│   └── ...
```
